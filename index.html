Excel Training — Web Data Sources

This page lists all courses in the Microsoft Excel training series. Where a course uses the Artifact Sequence Approach, links to the web-hosted demo and activity datasets are provided for direct import into Excel.

How to import a table into Excel:
Data tab  →  Get Data  →  From Other Sources  →  From Web  →  paste the data page URL  →  OK  →  select a table in the Navigator  →  Load  →  rename the worksheet to match the table name.
